Please Note:
  • Treemenu translation settings will override most, if not all, other translation options.
  • Treemenus usually present themselves in the default language.
  • The options Treemenu offers depend on the Vocabulary settings for translation.
  • For translation of content, edit the vocabulary (admin/content/taxonomy/) and set the 'Translation Mode' to 'Per language terms'.
For more help (or possibly confusion), see Translation.
